# Assesment Muman

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/nielsVoogt/assesment-muman.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the mock database
   ```sh
   npm run db
   ```
4. Start the application
   ```sh
   npm run dev
   ```
